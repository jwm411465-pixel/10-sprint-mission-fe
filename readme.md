## 🧩 프로젝트 소개
> 판다마켓(PandaMarket)은 반응형 웹 페이지 제작 연습을 위해 만든 쇼핑몰 랜딩 페이지입니다.

- 상단 고정 헤더(header)
- 2열 × 5행 그리드 상품 배치
- 하단 푸터(footer)
- 반응형 디자인 (1920px 이하에서 자동 정렬)
- SNS 아이콘 (페이스북, 인스타그램, 유튜브 등)
